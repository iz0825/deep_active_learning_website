Advantages of Elisa's Research

In Elisa’s research, she developed a probabilistic method which is utilised to estimate uncertainty as well as the quality of predicted segmentations. Her method leads to many advantages compared to the existing algorithms:

  • It is able to capture the full 3D content of medical scans
  • The algorithm is faster at the inference stage
  • The performance is better compared to the state-of-the-art algorithm in lung segmentation
  • It is able to estimate the quality of predicted segmentation for unseen data

Goal of the Project

The main focus of this project will be the way that active learning can be combined with deep learning methods.

However, integrating the two learning methods poses some problems. For instance, deep learning poses several difficulties when used in an active learning setting. Two main difficulties are outlined below.

  1. Need to be able to handle small amount of data.

    2. Recent advances in deep learning are notorious for their dependence on large amounts of data, but sometimes there is a need to process only small amount of data and the current scheme needs to be adjusted to do so.

  2. Most active learning acquisitions functions rely on model uncertainty, whereas in deep learning such model uncertainty is rarely represented. This is also the field that Elisa is currently working in as computer scientists are trying to find a robust solution to this problem by working on a way to estimate such uncertainty.
