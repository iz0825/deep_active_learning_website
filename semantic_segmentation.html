Overview

Semantic segmentation is a technique implemented in the field of deep learning, and hence the various fields of academia and industry which deep learning is applied to, in order to process images. More specifically, it encompasses an array of techniques for decomposing (segmenting) an image into its meaningful (semantic) parts. It provides a method that permits classification of the different components that make up a picture at the pixel level, since each pixel is allocated to its individual class (the component of the picture it belongs to). This means that the final product obtained when Semantic Segmentation is applied to an image is a version of the original image, that has clearly been divided into its components, allowing for meaningful deductions to be made about the information that is presented in the picture, alongside potential further processing.


Example - Patch Classification

An example of an early deep learning attempt at solving the problem is patch classification, a system by which each pixel in the image was classified into its respective class by analysis of the ’patch’ of image that formed the neighbourhood of pixels surrounding our given pixel. A drawback of this methodology is that the images were required to be fixed in size, which imposes limits on the applications of the technique. More recent mechanisms for semantic segmentation typically implement Convolutional Neural Network (CNN) architectures in order to carry out the task in a way that is not only exceedingly quicker than patch classification, but also has the advantage of allowing Semantic segmentation to be carried out to generate semantic maps for images of varying sizes, hence lifting some of the limitations of Patch classification.


Prospects

Overall, Semantic Segmentation provides a procedure by which we can computationally deduce important information presented in images, and therefore is applicable in a wide range of problems, including but not limited to autonomous vehicles, facial recognition, and of course, Medical Imaging
