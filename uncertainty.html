Why is there uncertainty in our results?

When using semantic segmentation in fields such as medical imaging, active learning is very beneficial as it means the most useful or informative information can be chosen to retrain the network and improve its performance. One such method by which this can be achieved is pixel-wise predictions so that the most useful pixels are then used to retrain the neural network. However, there is no way to verify that this method is robust and would indeed lead to the most useful pixels being chosen, and so there is a certain level of uncertainty that accompanies these predictions. How do we measure uncertainty?

In order to measure the uncertainty of the predictions resulting from the use of the active learning algorithm, it would be ideal if it were possible to analyse the entire distribution of the unlabeled data and the respective labels. However, if the fact that the algorithm would be repeatedly used were to be considered alongside the knowledge that there is usually no efficient method by which a whole distribution can be analysed, it would be reasonable to conclude that this approach would be far too time-consuming to be utilised in a practical environment. Thus, a concept known as variational inference is harnessed so that an approximation of the distribution can be made such that the amount by which this approximation diverges from the original full distribution is minimal.

Once this approximation has been made, the uncertainty for each pixel can be calculated such that a number of predictions are made regarding the same pixel, and the variation in the predictions coupled with other factors would determine the uncertainty of the result. The precision of this uncertainty estimate is also dependent on the number of the predictions whereby the larger the number, the higher the precision of the estimate. From this, a numerical score can be determined to estimate the prediction confidence.
